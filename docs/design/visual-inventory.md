## Direction

深石墨色的本地工作台。界面直接服务工具选择和执行，不使用营销页结构或默认卡片瀑布流。

## Tokens

- 背景：`#0a0e11`；表面：`#10161a`；分隔线：`#2e373b`。
- 强调色：`#c8f05a`；信息状态：`#60aefc`；错误状态：`#ff9e9e`。
- 大圆角仅用于主窗口与工具面板；列表保持窄边框和适中密度。
- 中英文均采用清晰的无衬线 UI 字体；代码编辑区使用等宽字体。

## Component Rules

- 左侧为固定工具导航，主视图为可搜索工具列表，右侧为最近使用轨道。
- JSON 工具使用输入/结果双编辑区，动作在标题下方，导出在底部。
- 图标为细线、圆角端点的代码原生图标；功能性文本与控件不可烘焙到图像中。
- 文件类工具须先选择路径并预览；批量重命名只输出副本，不改动源文件。
