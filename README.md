## 一个应用，处理日常琐事

OmniKit 面向需要频繁处理文本、文件与图片的日常工作。它将高频、实用的本地工具集中在统一工作台中，避免在网页、临时脚本和多个小软件之间来回切换。

处理过程在设备本机完成，文件和文本无需上传到陌生服务；界面保持简洁，让工具本身回到该有的位置。

## 现有工具

| 分类 | 工具 | 能做什么 |
| --- | --- | --- |
| 文本与编码 | JSON 格式化 | 格式化、校验与压缩 JSON 数据 |
| 文本与编码 | Base64 编解码 | 在文本、Base64 与 URL 编码之间转换 |
| 文本与编码 | 剪贴板历史 | 查找、置顶并再次复制最近记录的文本与链接 |
| 文件处理 | 文件哈希 | 计算 MD5、SHA-1、SHA-256 校验值 |
| 文件处理 | 批量重命名 | 先预览文件名，再输出保留原文件的副本 |
| 图片处理 | 图片压缩 | 压缩图片，或转换 JPG、PNG、WebP 格式 |
| 图片处理 | 截图识字 | 识别截图或本地图片中的中文、英文文字 |

## 为日常使用而设计

- **本地优先**：常用处理在桌面端完成，数据留在自己的设备中。
- **统一工作台**：分类、搜索、收藏与最近使用集中在一处。
- **轻量直接**：不堆叠复杂设置，打开工具即可开始处理。
- **持续更新**：应用可从 GitHub Releases 获取新版本与在线更新。
